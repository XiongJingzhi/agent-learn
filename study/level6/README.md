# Level 6：生产级能力与质量体系

## 学习目标
将 Agent 系统推进到生产可用：可观测、可部署、可治理、可持续迭代。

## 前置条件
- 完成 Level 5
- 具备基础 CI/CD 与容器化经验

## 核心主题
- 架构：高可用、可扩展、可审计
- 实现：日志、指标、追踪、安全与成本治理
- 学习方法：以 SLO/SLA 反推系统改造
- 测试：回归测试、稳定性测试、生产演练

## 实践任务
1. 设计观测面板（日志/指标/追踪）。
2. 建立 CI 流水线与质量门禁。
3. 定义安全与合规基线（密钥、权限、审计）。
4. 制定降级、熔断、限流与应急预案。

## 完成标准
- 关键链路可观测，异常可定位。
- 发布流程可重复，回滚路径明确。
- 质量门禁可阻断高风险变更。

## 常见误区
- 只做功能，不做观测与审计。
- 无容量/成本控制，运行风险累积。
- 把 CI 当执行器，忽略质量准入策略。
