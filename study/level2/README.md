# Level 2：任务规划型 Agent

## 学习目标
掌握规划-执行-反思模式，能对复杂目标进行分解、执行和动态调整。

## 前置条件
- 完成 Level 1
- 能实现基础记忆与日志记录

## 核心主题
- 架构：Planner / Executor / Reflector 分层
- 实现：任务分解、依赖排序、动态重规划
- 学习方法：先解释计划，再执行验证
- 测试：集成测试、状态流转测试、反思回路测试

## 实践任务
1. 选一个复杂任务并完成可执行任务树拆分。
2. 实现计划执行与失败回滚机制。
3. 增加执行日志并做一次反思优化。
4. 编写任务分解与状态流转测试。

## 完成标准
- 能输出可执行任务树并落地执行。
- 失败后可自动重规划或回滚。
- 测试可覆盖主流程和计划偏移场景。

## 常见误区
- 计划过细或过粗，执行效率低。
- 没有失败恢复策略，单点失败导致整体终止。
- 反思机制流于记录，不驱动改进。
