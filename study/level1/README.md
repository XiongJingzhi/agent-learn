# Level 1：LangGraph 单 Agent 基础

## 学习目标
掌握 LangGraph 单 Agent 的核心结构：状态、节点、边、工具调用、基本记忆。

## 前置条件
- 完成 Level 0
- 能运行 Python 项目与依赖安装

## 核心主题
- 架构：工具增强型单 Agent
- 实现：StateGraph、节点函数、条件边
- 学习方法：代码逐行讲解与对照复述
- 测试：节点单测、工具 mock、错误分支测试

## 实践任务
1. 实现一个含 2-3 个工具的单 Agent。
2. 增加条件路由与失败重试。
3. 记录一次“为什么这样设计”的架构说明。
4. 编写节点级和工具级测试用例。

## 完成标准
- 能独立搭建并运行单 Agent 工作流。
- 能清晰说明状态字段、边路由和工具选择逻辑。
- 测试覆盖主流程与至少 2 条异常路径。

## 常见误区
- 节点职责不清，导致流程难维护。
- 工具接口无约束，返回数据结构不稳定。
- 只测 happy path，不测失败路径。
