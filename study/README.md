## 如何学习
每个 Level 按固定顺序推进：
1. 阅读 `levelN/README.md` 明确目标与边界
2. 阅读 `levelN/PROGRESS.md` 执行阶段化计划
3. 学习 `notes/` 理解概念与设计权衡
4. 完成 `exercises/` 形成可验证实践结果
5. 完成 `projects/` 阶段项目并输出复盘
6. 对照 `checklists/completion.md` 验收
