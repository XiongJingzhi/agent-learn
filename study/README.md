# Agent 学习教程（study）

## 教程定位
这是当前仓库的正式学习教程目录，覆盖 Level 0 到 Level 7。
约束：
- 学习教程排除 `app/` 目录
- `app/` 仅作为综合案例参考（当前已有两个案例）
- 教程内容以可执行任务和可验证产出为核心

## 如何学习
每个 Level 按固定顺序推进：
1. 阅读 `levelN/README.md` 明确目标与边界
2. 学习 `notes/` 理解概念与设计权衡
3. 完成 `exercises/` 形成可验证实践结果
4. 完成 `projects/` 阶段项目并输出复盘
5. 对照 `checklists/completion.md` 验收

## Level 路径
- `level0`: Agent 认知地图
- `level1`: LangGraph 单 Agent 基础
- `level2`: 任务规划型 Agent
- `level3`: 框架对比与迁移
- `level4`: 高级单 Agent 架构
- `level5`: 多 Agent 协作系统
- `level6`: 生产级能力与质量体系
- `level7`: 前沿探索与综合项目

## 四条学习线
- 架构线：系统边界、抽象层次、设计权衡
- 实现线：核心代码、接口约束、工程落地
- 学习线：费曼解释、反思复盘、迁移能力
- 测试线：单测、集成、端到端、回归

## 全局文件
- `STYLE-GUIDE.md`：写作与术语规范
- `CURRICULUM-MAPPING.md`：源文档到教程章节映射
