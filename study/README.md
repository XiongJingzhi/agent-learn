# Agent 学习体系 v3（并行版）

## 1. 目标
本目录提供一套与根目录文档并行的学习体系，统一为 Level 0 到 Level 7。
设计原则：
- 不修改现有根目录学习文档
- 保留四条学习线：架构、实现、学习方法、测试
- 每个 Level 统一输出目标、前置条件、实践任务、完成标准、常见误区

## 2. 使用方式
建议每个 Level 按以下顺序推进：
1. 阅读 `levelN/README.md` 明确目标与边界
2. 在 `notes/` 编写概念笔记与费曼解释
3. 完成 `exercises/` 练习并记录结论
4. 在 `projects/` 完成阶段项目
5. 用 `checklists/` 做验收与复盘

## 3. 学习路径
- `level0`: Agent 思维与基础认知
- `level1`: LangGraph 单 Agent 基础
- `level2`: 任务规划型 Agent
- `level3`: 框架对比与选择（含 OpenClaw/OpenCode）
- `level4`: 高级单 Agent 架构
- `level5`: 多 Agent 协作系统
- `level6`: 生产级能力与质量体系
- `level7`: 前沿探索与综合项目

## 4. 角色映射
- Architect：架构演进、设计权衡、系统边界
- Senior Dev：框架实践、工具开发、工程实现
- Feynman Mentor：i+1、类比学习、解释验证
- Testing Expert：单测/集成/端到端与质量评估

## 5. 全局文档
- `STYLE-GUIDE.md`：命名、术语与写作模板
- `CURRICULUM-MAPPING.md`：源文档到 v3 的可追溯映射
