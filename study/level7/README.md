# Level 7：前沿探索与综合项目

## 学习目标
形成“研究 -> 验证 -> 工程化”闭环，完成可展示、可复用的综合 Agent 项目。

## 前置条件
- 完成 Level 6
- 具备跨模块整合能力

## 核心主题
- 架构：前沿方向评估与路线规划
- 实现：综合项目设计、里程碑管理、复盘迭代
- 学习方法：假设驱动实验 + 费曼输出
- 测试：实验可复现性与结果可解释性

## 与 app 目录的关系
- 本学习计划明确排除 `app/`。
- `app/` 当前作为综合案例区（已有两个案例），可作为结果对照或灵感来源，但不作为本目录必修内容。

## 实践任务
1. 选定一个前沿主题并形成研究假设。
2. 完成一个综合项目的设计与落地文档。
3. 输出“架构决策记录 + 风险清单 + 改进计划”。
4. 形成可复现实验步骤与评估指标。

## 完成标准
- 项目具备清晰边界、架构说明和可验证指标。
- 实验结论可复现、可解释、可比较。
- 能输出下一轮演进路线。

## 常见误区
- 只追新概念，缺少工程落地与评估。
- 缺少决策记录，导致方案不可复盘。
- 未定义指标，无法判断成效。
